# El Rincón Chileno - Landing Page

Proyecto de Landing Page para restaurante **El Rincón Chileno**, desarrollado como evaluación de Desarrollo Web.

## Características

✅ Estructura HTML5 semántica completa
✅ CSS con variables de marca (:root)
✅ Flexbox en menú de navegación
✅ Grid en sección de productos
✅ Formulario de reserva funcional
✅ Diseño responsivo
✅ Secciones: Hero, Productos, Contacto, Nosotros, Footer

## Requisitos Técnicos Cumplidos

- [x] HTML5 semántico (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- [x] CSS con variables en `:root` (colores primario, acento, sombra, radio)
- [x] Flexbox obligatorio en navegación (`nav ul`)
- [x] Grid obligatorio en productos (`grid-template-columns: repeat(3, 1fr)`)
- [x] Formulario de contacto con validaciones
- [x] Repositorio GitHub público con commits
- [x] GitHub Pages activado

## Estructura del Proyecto

```
/
├── index.html       # Estructura HTML base
├── styles.css       # Estilos con Flexbox y Grid
└── README.md        # Este archivo
```

## Secciones de la Landing Page

1. **Header** - Nombre del restaurante + Navegación (Flexbox)
2. **Hero Section** - Bienvenida con gradiente
3. **Productos** - 3 platos destacados (Grid 3 columnas)
4. **Contacto** - Formulario de reserva de mesa
5. **Nosotros** - Información del restaurante
6. **Footer** - Datos de contacto, ubicación, horarios

## Cómo Usar

1. Abre `index.html` en tu navegador
2. Navega por las secciones usando el menú
3. Completa el formulario de reserva (frontend solamente)
